# AGENTS.md - Project Guidelines

## Prohibiciones Obligatorias

### ❌ NO crear documentación adicional
- No crear archivos README.md innecesarios
- No crear archivos de documentación extra
- No crear guías de uso
- No crear archivos de notas o comentarios

### ❌ NO crear archivos de texto innecesarios
- No crear archivos .txt
- No crear archivos de configuración de documentación
- No crear archivos de registro o logs innecesarios
- No crear archivos temporales o de prueba

### ❌ Regla General
**Solo crear código funcional y archivos esenciales para que el proyecto funcione.**

## Comandos de Código
Crear solo lo necesario para implementar funcionalidad real.
